Certain modules are required in index_user for proper functioning------

1>Express

Purpose: Fast, unopinionated, minimalist web framework for Node.js.
Installation Command: npm install express

2>Body-parser

Purpose: Node.js body parsing middleware.
Installation Command: npm install body-parser

3>Cors

Purpose: Cross-Origin Resource Sharing middleware for Express.
Installation Command: npm install cors

4>Swagger-jsdoc

Purpose: Generates Swagger/OpenAPI specification from JSDoc comments.
Installation Command: npm install swagger-jsdoc

5>Swagger-ui-express

Purpose: Middleware to serve Swagger UI generated by swagger-jsdoc.
Installation Command: npm install swagger-ui-express

6>Moment-timezone

Purpose: Date and time manipulation with timezone support.
Installation Command: npm install moment-timezone
